Learning the fundamentals of Fusion 360

This simple design focuses on training my skills in Autodesk Fusion. It includes the process of creating base, how to navigate the interface, and how to use basic tools like sketching, extruding, and assembling components.

I followed the tutorial on the foundation of fusion 360(link).

First, I created the baseplate using the sketching tool. This involved drawing the outline of the baseplate in 2D with the correct dimensions. As of now, this baseplate is only 2 dimensional.

Next, I used the extrude tool to give the baseplate some thickness, turning it into a 3D object, giving it a circular holes.

After I was done with the baseplate, I created the brackets: I started with sketching the base of the bracket in 2D.

After sketching the base, I used the extrude tool to give the bracket some thickness, turning it into a 3D object.

Next, I sketched the wall of the bracket in 2D.

Next, I used the extrude tool to give the wall thickness and height.

Next, I used the sketch tool to create circles on the wall, which will later become holes.

Next, I used the extrude tool in to the wall giving holes depth.

Next, I used the fillet tool to round the edges of the holes.

Finally, the bracket part is done.

For the final part that I'm making, I created a bolt. I started with sketching the profile of the bolt in 2D and extruding it.

Next, I used the sketch tool to give the bolt a head.

Next, I used the extrude tool to give the bolt head some thickness.

Next, I used the chamfer tool to create a beveled edge on the bolt head and the end of the body, so that the screws are able to fit into the holes of the bracket.

Next, I used the sketch tool to create a keyhole shape on the bolt head.

Finally, I used the extrude tool to give the keyhole shape some depth, so that it can accommodate an allen key.

FINALLY, I used the assemble tool to put all the parts together. I used the joint tool to connect the baseplate and the brackets, and then connected the bolt to the bracket.
